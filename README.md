# Mate

Mate extends the native JavaScript/CoffeeScript, and works on browsers and Node.js:

- A new way of handling events
- Array extensions
- Advanced timers
- Observer
- "Functional" functions
- 2D point & complex math
- ECMAScript 6th features
- Very natural testing
